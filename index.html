Cortex provides end-to-end solutions for front-end development, designed to change the way you write web modules.

Getting Started

Write Browser Code like Node.js

With cortex, you could write web modules exactly the way you do with node.js. You could get browser versions of the node core libraries events, stream, path, url, assert, buffer, util, querystring, http, etc when you require() them.

Social Coding

You never fight alone ! You could collaborate and share with your teammates and even talented people all over the world.

Awesome Experience for Debugging

You could start the local debugging server with ONE single command. Say goodbye to Fiddler and Charles.

A Fusion with Server-side bundles

Cortex provides a way to make front-end packages work much better with server-side modules by decoupling each others.
